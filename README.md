## Descripción

Este repositorio recopila implementaciones, análisis y buenas prácticas para el modelado de **series de tiempo**, con énfasis en **modelos ARIMA** (AutoRegressive Integrated Moving Average) y **técnicas de descomposición** (aditiva y multiplicativa). Sirve como referencia para futuros proyectos de pronóstico, documentando desde la identificación de componentes (tendencia, estacionalidad, ruido) hasta la validación y diagnóstico de modelos.

---

## Objetivos

- **Documentar** la teoría y práctica de modelos ARIMA.
- **Implementar** descomposición clásica (additive/multiplicative) y descomposición STL.
- **Proporcionar** flujos de trabajo reproducibles para análisis de series temporales.
- **Servir como referencia** para seleccionar el modelo adecuado según las características de la serie.
- **Incluir** diagnósticos de residuos, validación cruzada temporal y métricas de error.
